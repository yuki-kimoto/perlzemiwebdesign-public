【HTML5】trタグ - テーブルの行を作成する

trタグでテーブルの行を作成できます。 

<tr>
</tr>

一般的には、列と項目を作成するために、trタグとtdタグをtableタグの下に配置します。

1行1列1行2列1行3列
2行1列2行2列2行3列 
<style>
  table {
    border-collapse: collapse;
  }

  table td {
    border:1px solid #ddd;
    padding:7px;
  }
</style>

<table>
  <tr>
    <td>1行1列</td>1行2列<td><td>1行3列</td>
  </tr>
  <tr>
    <td>2行1列</td>2行2列<td><td>2行3列</td>
  </tr>
</table>
